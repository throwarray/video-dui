## Broadcast from OBS to FiveM (GTAV)

# INSTALL

```
start video-stream
```

- Publish stream with `OBS` to RTMP
```
Stream Type: Custom Streaming Server
URL: rtmp://localhost/live
Stream key: STREAM_NAME?secret=secret
```

Type `/video-stream` to start streaming
